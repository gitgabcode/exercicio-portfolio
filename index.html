Meus projetos

Cyber Virus History

Cyber Virus History é um site educacional que explora a evolução dos vírus de computador, desde o Creeper até ameaças como LoveLetter e Heartbleed. Desenvolvido por Gabriel Marcelino, o site apresenta seções sobre vírus importantes, um vídeo explicativo e links para aprofundamento, usando HTML semântico e CSS com variáveis para uma estilização acessível e fácil de manter.

Site de Notícias com Áudio Simples

O projeto Site de Notícias com Áudio Simples é uma página de notícias que inclui um recurso de áudio para acessibilidade. Desenvolvido com HTML e CSS, o site utiliza estrutura semântica para facilitar a navegação e apresenta um design simples e funcional, com opções para escutar o conteúdo das notícias em áudio.
